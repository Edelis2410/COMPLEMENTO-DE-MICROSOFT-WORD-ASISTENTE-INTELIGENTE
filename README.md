**Complemento para Microsoft Word basado en Modelos de Lenguaje Extensos (LLM)**

---

## Descripción

Asistente de Comprensión Inteligente es un complemento para Microsoft Word que permite a estudiantes y profesionales comprender terminos complejos en documentos academicos sin salir del entorno de edicion.

1. **Selecciona** una palabra o frase
2. **Clic derecho** → "Consultar"
3. **Obtén** una explicación clara generada por inteligencia artificial

---

## Características

- Menu contextual - Consulta desde el clic derecho
- IA integrada - Explicaciones generadas por LLM
- Tres niveles - Sencilla, Técnica o Practica

---

## Instalación

1. Descarga el archivo `manifest.xml` del repositorio
2. Abre Microsoft Word Online
3. Ve a `Insertar` → `Complementos` → `Complementos de Office`
4. Selecciona `Subir mi complemento`
5. Elige el archivo `manifest.xml`
6. Listo. El complemento aparecera en tu pestaña de inicio

---

## Tecnologías

- Office JavaScript API
- Node.js
- Modelos de Lenguaje Extensos (LLM)
- HTML5, CSS3, JavaScript
